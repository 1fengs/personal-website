# 🚀 Personal Portfolio Website

This is my personal portfolio website, built to showcase my projects, skills, and experience as a software engineer with a focus on robotics, embedded systems, and full-stack development.

## 🧑‍💻 About Me

I am a software engineer based in Berlin with experience in:

- Autonomous driving (ROS2, MPC, C++)
- Test automation and validation
- Embedded systems and robotics
- Web development

This website serves as a central place to present my work, technical skills, and professional journey.

---

## 🛠️ Tech Stack

- Frontend: Next.js, React, TypeScript
- Styling: Tailwind CSS
- Animation: Framer Motion
- Icons: Lucide React

---

## ✨ Features

- Project showcase with detailed modal view
- Structured skill visualization with categories
- Modern UI with responsive design
- Smooth animations and transitions
- GitHub & live demo links integration

## ⚙️ Getting Started

### 1. Clone the repository

git clone https://github.com/1fengs/personal-website.git  
cd personal-website

### 2. Install dependencies

npm install

### 3. Run the development server

npm run dev

### 4. Open in browser

http://localhost:3000

---

## 🧩 Configuration

Make sure you are using Node.js version: v24.14.0
